# Pequena Homenagem — site-romantico

Estrutura inicial criada com HTML, CSS, JS e um arquivo de mensagens.

Como usar:

1. Coloque imagens em `assets/images/` (ex.: `capa.jpg`, `foto1.jpg`).
2. (Opcional) coloque `musica.mp3` em `assets/audio/` e adicione um player.
3. Abra `index.html` no navegador.

Arquivos principais:

- `index.html` — estrutura das telas
- `assets/css/` — estilos e animações
- `assets/js/` — scripts (`main.js`, `typing.js`, `hearts.js`)
- `data/messages.js` — mensagens exibidas no carrossel
